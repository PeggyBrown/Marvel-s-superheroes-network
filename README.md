# Marvels-superheroes-network
Marvel's superheroes create network, which behaves similarly to human network. 
They create 6 main clusters which are gathered around: Avengers, X-Men, Peter Parker/Spider-Man, X-Men and Fantastic 4 (Invisible Woman, Mr. Fantastic, Human Torch, and Thing).

More info about Marvel's network behavior can be found here: https://www.dropbox.com/s/tamwvjij3ywkr1k/Marvel%20Worldwide%20Inc.docx?dl=0
